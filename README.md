# A Novel Contrastive Loss for Zero-Day Network Intrusion Detection

Official Github repo for the paper "A Novel Contrastive Loss for Zero-Day Network Intrusion Detection". This repository covers a reference implementation and scripts to reproduce experimental results for both the Contrastive Learning for Anomaly Detection (CLAD) and Contrastive Learning for Open Set Recognition (CLOSR) methods decribed in this work.
